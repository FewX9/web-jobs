คำอธิบาย

หลักสูตรนี้คุณจะได้เรียนรู้การประยุกต์ใช้ภาษาไพธอนสำหรับนำการสร้างแอพพลิเคชั่นสำหรับใช้งานจริง

ในหลักสูตรนี้จะเป็นการนำเอาความรู้ภาษาไพธอนมาประยุกต์ใช้งานร่วมกับ ไลบรารี่/ Package เสริมอื่นๆ เช่น:

  • จัดการฐานข้อมูล (Database)
  • การสร้างคิวอาร์โค้ด
  • แปลภาษาโดยใช้ Google Translate
  • การดาวน์โหลดวิดีโอ
  • การเขียนโปรแกรมเชิงวัตถุ (OOP) สำหรับสร้างแอพทำข้อสอบ
  • การพัฒนาเว็บฝั่ง Backend ด้วย Flask Framework , Django Framework
  • การทำ Web Scraping ด้วย BeautifulSoup4

เหมาะสำหรับผู้ที่มีพื้นฐานการเขียนโปรแกรมด้วยภาษา Python (ไวยากรณ์พื้นฐาน , โครงสร้างข้อมูล)

ผู้เรียนต้องมีพื้นฐาน HTML5 , CSS3 , JavaScript และ Bootstrap 5 เพื่อใช้สำหรับนำมาสร้างเว็บแอพพลิเคชั่น

หลักสูตรนี้เหมาะกับผู้ที่สนใจอยากจะพัฒนาทักษะด้านการเขียนโปรแกรมด้วยภาษา Python
